Folha de Estilo (CSS)
descendentes "div p" o seletor "pai filho" seleciona todos os elementos filho que são descendentes do elemento pai, independentemente de sua posição na árvore de elementos
filhos "div > p" o seletor "pai > filho" seleciona todos os elementos filho que são diretamente descendentes do elemento pai
adjacentes "div ~ p" o seletor "irmão1 ~ irmão2" seleciona todos os irmãos irmão2 que seguem o irmão1
imediatamente adjacentes "div + p" o seletor "irmão1 + irmão2" seleciona o irmão2 apenas se ele é um irmão adjacente do irmão1
agrupar (separar por virgulas) "div,p{color: red}" o seletor "pai filho" seleciona todos os elementos filho que são descendentes do elemento pai, independentemente de sua posição na árvore de elementos
